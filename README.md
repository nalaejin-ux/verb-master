# 💖 동사 마스터 — 불규칙 동사 100개 완전정복!

초등학생과 함께 영어 불규칙 동사를 재미있게 외울 수 있는 웹 앱입니다.  
아이패드, 아이폰, PC 어디서든 사용 가능하고, 홈 화면에 추가하면 앱처럼 쓸 수 있어요! 📱

---

## 🌐 바로 사용하기

👉 **https://[내 깃허브 아이디].github.io/verb-master**

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 🎯 퀴즈 | 현재형을 보고 과거형·과거분사형을 **직접 입력**해야 통과! |
| 🟣 SKIP | 모르면 건너뛰고 **모르는 단어 그룹**에서 나중에 집중 연습 |
| 🔥 오답 집중 | 틀린 단어만 모아서 반복 연습 |
| 🔍 동사 찾기 | 카드를 **탭하면** 과거형·과거분사형 즉시 확인 |
| 📋 전체 단어표 | 100개 전체 보기 + 검색 |
| 💾 진행 저장 | 점수·오답·모르는 단어 자동 저장 (브라우저 닫아도 유지) |
| 📱 앱 설치 | 홈 화면에 추가해서 앱처럼 사용 가능 |
| ✈️ 오프라인 | 한 번 열면 인터넷 없이도 사용 가능 |

## 🚀 GitHub Pages 배포 방법 (5분 완성!)

### 1단계 — GitHub 계정 만들기
아직 없다면 → **https://github.com** 에서 무료 가입

### 2단계 — 새 저장소(Repository) 만들기

1. GitHub 로그인 후 오른쪽 위 **`+`** 버튼 클릭
2. **New repository** 선택
3. Repository name: **`verb-master`** 입력
4. **Public** 선택 (꼭! Private이면 Pages가 안 됨)
5. **Create repository** 클릭

### 3단계 — 파일 올리기

저장소 페이지에서:
1. **uploading an existing file** 링크 클릭 (또는 Add file → Upload files)
2. 이 폴더의 파일들을 **모두 드래그 앤 드롭**:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. 아래쪽 **Commit changes** 버튼 클릭

### 4단계 — GitHub Pages 켜기

1. 저장소 상단 **Settings** 탭 클릭
2. 왼쪽 메뉴에서 **Pages** 클릭
3. **Source** → **Deploy from a branch** 선택
4. **Branch** → **main** 선택, 폴더는 **/ (root)** 유지
5. **Save** 클릭
6. 1~2분 후 페이지 새로고침하면 주소가 나타남!

```
🎉 완성! 주소: https://[내깃허브아이디].github.io/verb-master
```

## 📱 아이패드/아이폰에서 앱처럼 설치하기

1. **Safari**로 위 주소 열기 (크롬은 안 됨!)
2. 하단 **공유 버튼** (□↑) 탭
3. **"홈 화면에 추가"** 탭
4. **추가** 탭
5. 홈 화면에 💖 아이콘이 생겼어요!

## 📖 사용 방법

```
홈 화면
├── 🎯 순서대로     → 1번부터 10개씩, 과거형·과거분사형 입력
├── 🟣 모르는 단어  → SKIP한 단어 집중 연습
├── 🔥 오답 집중    → 틀린 단어 집중 연습
├── 🔀 랜덤 10개    → 무작위 10문제
└── 🔍 동사 찾기   → 카드 탭하면 변화형 즉시 확인!
```
